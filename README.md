# Rushabh - AI Video Caption Generator

A Next.js web application that allows users to upload videos and generate captions automatically.

## Features

- Upload video
- Auto-generate captions
- Edit subtitles
- Download captioned video
- Video summarization
- Clean UI

## Tech Stack

- Next.js
- React
- Tailwind CSS
- AWS Transcribe
- FFmpeg (WASM)
